# 1、网站制作流程

解释一下其中的步骤

第四步**初稿审核**

首先需要制作**原型图**(下面有示例)，通过工具知道网站完成的大致样式

然后美工制作**psd效果图**，然后跟客户对接审核，定下初稿

第七步**上线培训**，就是教一下他们我们Web的架构以及网页的使用等等

第八步**后期维护**，维护一下bug或者加点小版块(大板块得加钱)

By the way, 项目经理就是负责把控整个流程的
